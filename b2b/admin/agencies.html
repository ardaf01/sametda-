Yeni Acente Ekleme

Supabase mimarisi gereği acenteleri kendi şifrelerini belirleyebilecekleri şekilde en güvenli olarak panelden davet edebilirsiniz.

Adımlar:

  1. Supabase Paneline girin.
  2. Authentication -> Users sekmesine gidin.
  3. "Add User" -> "Create User" butonuna tıklayın.
  4. Acentenin E-Postasını ve geçici şifresini yazın.
  5. User Metadata alanına şunları yazarak kaydedin: 
    {
  "company_name": "Sunny Tours",
  "contact_person": "Hans",
  "phone": "+49 xxx"
}
  6. Acente anında bu listeye düşecek ve giriş yapabilecektir.
